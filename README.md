# Frontend SGBE

Projeto frontend simples para gerenciamento de biblioteca escolar (SGBE).

Visão geral

- Interface estática em HTML/CSS/JavaScript para navegação entre páginas de alunos, livros e empréstimos.

Estrutura do repositório

- `index.html` - Página inicial
- `alunos.html` - Gestão de alunos
- `livros.html` - Catálogo de livros
- `emprestimos.html` - Registro de empréstimos
- `app.js` - Scripts de interação
- `global.css`, `alunos.css`, `livros.css`, `emprestimos.css`, `aluno.css` - Estilos

Como executar

1. Abrir `index.html` em um navegador moderno (Chrome, Firefox, Edge).
2. Não há servidor necessário para uso local; carregar o arquivo diretamente no navegador é suficiente.
